How to use this file:
1. Run a Google Keyword Planner pass on every cell in the matrix below.
2. Drop the monthly search volume + competition score into the cell.
3. Star (⭐) the winning phrase per page — the one we write `<title>`, H1, slug, and schema `name` for.
4. Mark page status: **EXISTS** (already deployed), **STUB** (route reserved, no content), **NEW** (to build).
5. Commit. The rest of the plan compiles against this file.

> **Ronald's rule (step 1):** Don't guess what customers call it. "Lap siding" vs "exterior siding." For us: "exercise log" vs "workout tracker" vs "fitness journal" all behave differently in volume. Trust the data, not intuition.

## Modifier vocabulary (mix into titles, H2s, and FAQ questions)

Use these where natural. Never stuff. Modifiers that perform well in this niche:

- **Intent:** `free`, `no app`, `without an app`, `online`, `easy`, `simple`, `best`, `2026`
- **Action verbs:** `track`, `log`, `count`, `monitor`, `record`, `journal`
- **Audience:** `for beginners`, `for women`, `for men`, `for runners`, `for lifters`, `for cutting`, `for bulking`
- **Comparison:** `vs MyFitnessPal`, `vs Cronometer`, `vs Lose It`, `vs Strava`, `alternative to`
- **Format / device:** `on iphone`, `on android`, `on mac`, `on pebble`, `on apple watch`, `on garmin`, `on oura`
