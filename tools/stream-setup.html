Export for OBS / Streamlabs

Two ways to use these overlays in OBS:
A) PNG layers. Download each overlay as a transparent PNG and add as image source in OBS. Quick but static.
B) Browser source (recommended). Add a Browser Source in OBS pointing at this page with ?obs=<overlay> — the overlay renders live with transparent background, so counters & leaderboard keep animating.
